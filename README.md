# رنيم — Arabic AI Discord Bot

بوت ديسكورد ذكي باللغة العربية مبني بـ Discord.js + Express + Groq AI

## المميزات
- 🤖 دردشة بالذكاء الاصطناعي (Groq / Llama 3.3)
- 🎨 توليد الصور (/image) بـ Runware.ai
- 🔨 أوامر إدارة (/mod)
- 📝 ملاحظات وتذكيرات
- 💾 ذاكرة دائمة
- 🔄 Keep-alive على بورتين
- 🖥️ لوحة تحكم ويب لتشغيل/إيقاف البوت

## المتطلبات
- DISCORD_BOT_TOKEN
- GROQ_API_KEY
- RUNWARE_API_KEY

```bash
pnpm install
pnpm --filter @workspace/api-server run dev
```
